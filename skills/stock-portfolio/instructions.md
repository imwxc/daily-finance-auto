# stock-portfolio Skill - 股票持仓分析

## 技能描述

分析股票持仓，计算收益，生成周报/月报。

## 触发条件

- 定时触发：周五下午 4 点（收盘后）
- 关键词触发：股票、持仓、周报

## 执行步骤

### 1. 采集最新数据

```bash
cd ~/daily-finance-auto
source .venv/bin/activate
python -m src.main collect --type stock
```

### 2. 查看持仓

```bash
python -m src.main stocks
```

### 3. 生成报告

生成持仓分析报告，包含：

- 持仓收益
- 技术指标分析
- 市场风险评估
- 操作建议

## 示例对话

**用户**: 生成这周的股票持仓报告

**AI**: 
📊 本周持仓报告 (2026-02-22 ~ 2026-02-28)

| 股票 | 持仓 | 成本 | 现价 | 收益率 |
|------|------|------|------|--------|
| 贵州茅台 | 100股 | 1800.00 | 1850.00 | +2.78% |
| 平安银行 | 500股 | 12.50 | 11.80 | -5.60% |

总收益率: -1.41%

💡 分析:
- 贵州茅台走势良好，建议持有
- 平安银行跌破成本价，关注 11.50 支撑位

## 注意事项

- 需要配置 Tushare Token
- 股票分析仅供参考，不构成投资建议
