# fund-dca Skill - 基金定投监控

## 技能描述

监控用户持有的基金，分析定投信号，生成报告并发送通知。

## 触发条件

- 定时触发：工作日早上 9 点
- 关键词触发：定投、基金、DCA

## 执行步骤

### 1. 采集最新数据

首先采集最新的基金净值数据：

```bash
cd ~/daily-finance-auto
source .venv/bin/activate
python -m src.main collect --type fund
```

### 2. 检查定投信号

分析所有监控基金的定投信号：

```bash
python -m src.main invest
```

### 3. 生成报告

根据分析结果生成报告：

**均线偏离策略信号：**

| 信号 | 条件 | 操作 |
|------|------|------|
| 📉 加仓 | 净值 < 均线 10% | 加倍定投 |
| ✅ 正常 | 净值在均线附近 | 正常定投 |
| 📈 减仓 | 净值 > 均线 10% | 减半定投 |
| ⏸️ 跳过 | 净值 > 均线 20% | 跳过本次 |

### 4. 发送通知（可选）

如果配置了通知渠道，发送定投信号报告：

```bash
python -m src.main invest --notify
```

## 配置要求

1. 在 `config.yaml` 中配置监控基金：
```yaml
funds:
  - code: "000001"
    name: "华夏成长"
    amount: 1000
  - code: "110022"
    name: "易方达消费"
    amount: 1000
```

2. 配置通知渠道（可选）：
```yaml
notify:
  wechat:
    enabled: true
    send_key: "YOUR_SEND_KEY"
```

## 注意事项

- 数据采集依赖 AKShare，首次运行可能需要安装依赖
- 定投信号仅供参考，不构成投资建议
- 建议结合个人风险偏好和资金状况做决策
