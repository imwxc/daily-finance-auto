# 数据 API 选型调研

## 一、免费数据源

### 1. AKShare（推荐）

| 属性 | 详情 |
|------|------|
| **GitHub** | https://github.com/akfamily/akshare |
| **文档** | https://akshare.akfamily.xyz/ |
| **费用** | 完全免费 |
| **覆盖范围** | 股票、基金、期货、外汇、债券、宏观经济 |

```python
import akshare as ak

# 获取基金净值
df = ak.fund_open_fund_info_em(fund="000001", indicator="单位净值走势")

# 获取 A 股日线
df = ak.stock_zh_a_hist(symbol="000001", period="daily")
```

**优势**：
- 完全免费开源
- 接口丰富（78 个专业工具，190+ 数据接口）
- 中文文档完善
- 覆盖另类数据（宏观指标、行业数据）

**劣势**：
- 基于爬虫，稳定性差
- 延迟不可控
- 不适合生产环境高频调用

### 2. TuShare（免费版）

| 属性 | 详情 |
|------|------|
| **GitHub** | https://github.com/waditu/tushare |
| **注册** | https://tushare.pro/register |
| **免费额度** | 120 积分（注册即送） |

**劣势**：免费版功能有限，推荐升级 Pro 版

### 3. 新浪/腾讯财经 API

非官方接口，随时可能失效，仅适合个人学习。

## 二、付费专业数据源（生产环境推荐）

### 1. Tushare Pro（强烈推荐 ⭐⭐⭐⭐⭐）

| 属性 | 详情 |
|------|------|
| **官网** | https://tushare.pro/ |
| **基础套餐** | ¥500 = 5000 积分（一次性） |
| **分钟级数据** | 额外约 ¥1000/月 |

```python
import tushare as ts

ts.set_token('your_token_here')
pro = ts.pro_api()

# A 股日线数据
df = pro.daily(ts_code='000001.SZ', start_date='20240101', end_date='20240214')

# 基金净值
df = pro.fund_nav(ts_code='000001.OF')
```

**优势**：
- A 股基本面数据工业标准
- 数据清洗质量高
- API 稳定可靠
- 5000 积分后几乎无限制
