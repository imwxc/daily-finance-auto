## 八、风险提示

1. **脉冲行情特征明显** — 地缘冲突引发的上涨往往是短期脉冲，需警惕回调
2. **冲突若缓和，油价迅速回落** — 若伊朗恢复生产或谈判达成，油价可能回调
3. **黄金ETF近期波动大** — 短期波动大，建议逢低布局而非追涨
4. **A股受影响有限** — 沪指3316点缺口有支撑，中长期慢牛格局未变

---

## 九、核心原则

1. **油气是短线品种** — 冲突缓和即卖出，不恋战
2. **军工是中线品种** — 可置换到科技，保留部分底仓
3. **黄金是长线品种** — 地缘逻辑只是加分项，核心是去美元化
4. **宁可少赚，不可亏损** — 到止盈线先卖一半锁定利润
5. **新闻比技术更重要** — 地缘资产对新闻极其敏感
